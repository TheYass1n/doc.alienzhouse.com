# إعدادات بطاقة المنتج في قالب لوكس

بطاقة المنتج هي واجهتك الأولى مع العميل في صفحات المنتجات أو القوائم، ولوكس يمنحك تحكمًا كاملاً في تفاصيلها:

## 1. ألوان البطاقة
- لون خلفية البطاقة
- لون نص البطاقة
- لون خلفية زر الشراء
- لون نص زر الشراء
- لون خلفية صورة المنتج (مفيد للصور بدون خلفية)
- لون عنوان المنتج

## 2. إعدادات العنوان
- عدد أسطر العنوان: اختر إذا كنت تريد العنوان في سطر واحد أو أكثر.
- سمك الخط: اختر بين نحيف، عادي، متوسط أو سميك.

## 3. الشكل والحجم
- انحناء زوايا البطاقة: خصص درجة الانحناء لتناسب هوية متجرك.
- حجم صورة البطاقة: مربع، عمودي، مستطيل أو واسع.
- المساحات بين البطاقات: تحكم في المسافة بين كل بطاقة وأخرى.

## 4. خيارات متقدمة
- عرض قيم حقل الخيارات كقائمة منسدلة أو أزرار اختيار (اختيار واحد أو عدة اختيارات).
- تخصيص صور توضيحية حسب نوع الخيارات.

---

كل هذه الخيارات تمنحك مرونة عالية لتصميم بطاقات منتجات عصرية وجذابة تعكس احترافية متجرك وتُسهل على العميل اتخاذ قرار الشراء.
