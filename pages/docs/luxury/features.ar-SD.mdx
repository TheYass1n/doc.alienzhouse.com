# ميزات ثيم لاكشري

ثيم لاكشري يوفر مجموعة واسعة من خيارات التخصيص لتناسب جميع العلامات التجارية الجريئة.

## ملخص الخيارات الرئيسية:

- **اعدادات عامة:** تحكم في زوايا العناصر والأزرار وحقول الإدخال.
- **بطاقة المنتج:** تخصيص شكل ومحتوى بطاقة المنتج بالكامل.
- **صفحة المنتج:** خيارات متقدمة لعرض المنتج والعروض والحقول المخصصة.
- **شريط التنبيهات المتحرك:** عرض إعلانات أو تنبيهات متحركة في أعلى المتجر.
- **صفحة التصنيف:** تحكم في طريقة عرض المنتجات داخل التصنيفات.
- **إعدادات PWA:** اجعل متجرك يعمل كتطبيق ويب احترافي.
- **شاشة البداية:** تخصيص شاشة البداية عند فتح المتجر.
- **زر الاتصال السريع:** تسهيل تواصل العملاء معك مباشرة.

للمزيد من التفاصيل حول كل خيار، راجع قسم [خيارات الثيم](./theme-options/store-options.ar-SD.mdx) وروابط الخيارات الفرعية.

---

> ⚡ كل خيار موثق بشكل مفصل في صفحة خاصة به ضمن قسم خيارات الثيم.

# Luxury Theme Features

Luxury is highly customizable for bold brands.

## Hero Banner Customization

Change the hero banner background:

```css
#luxury-hero {
  @apply bg-gradient-to-r from-gold to-black;
}
```

## Typography
Adjust the parallax speed in the relevant section’s JavaScript or component props.
