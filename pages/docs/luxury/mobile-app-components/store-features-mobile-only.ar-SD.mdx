# خصائص المتجر (تطبيق الجوال فقط)

مكون "خصائص المتجر (تطبيق الجوال فقط)" يتيح لك إبراز أهم مميزات أو خصائص متجرك في تطبيق الجوال، من خلال عرض صور وعناوين ووصف مختصر لكل ميزة بشكل أنيق وجذاب.

## المميزات الرئيسية

- **إضافة حتى 3 خصائص:**
  - يمكنك إضافة حتى ثلاث خصائص أو مميزات رئيسية لمتجرك، مع صورة وعنوان ووصف لكل ميزة.
- **تخصيص الألوان:**
  - خصص لون النص لكل ميزة على حدة، واختر لون خلفية موحد للقسم بالكامل.
- **صور مخصصة:**
  - يمكنك رفع صورة لكل ميزة (يفضل مقاس 200×200 بكسل) لتوضيح الفكرة بشكل بصري.

## نصائح عملية
- اختر أهم 3 مميزات تميز متجرك عن المنافسين ووضحها بشكل مختصر وجذاب.
- استخدم صورًا واضحة وملونة لكل ميزة لجذب الانتباه.
- اختر لون خلفية متناسق مع ألوان التطبيق ليظهر القسم بشكل احترافي.
- لا تكثر من النصوص، واجعل العناوين مختصرة وسهلة الفهم.
