# قسم الأسئلة الشائعة الرئيسية (تطبيق الجوال فقط)

مكون "قسم الأسئلة الشائعة الرئيسية (تطبيق الجوال فقط)" يتيح لك عرض أهم الأسئلة الشائعة مع إجاباتها في الصفحة الرئيسية لتطبيق الجوال، مع إمكانية إضافة فيديو وصورة توضيحية وخلفية ملونة.

## المميزات الرئيسية

- **عنوان ووصف القسم:**
  - أضف عنوانًا رئيسيًا ووصفًا مختصرًا لجذب انتباه المستخدمين وتوضيح أهمية القسم.
- **لون الخلفية:**
  - خصص لون خلفية القسم ليتناسب مع هوية متجرك.
- **صورة وفيديو توضيحي:**
  - يمكنك إضافة صورة مصغرة وفيديو يوتيوب لشرح أو توضيح فكرة معينة.
- **إضافة حتى 3 أسئلة شائعة:**
  - أضف حتى 3 أسئلة شائعة مع إجاباتها، لتسهيل حصول المستخدمين على المعلومات المهمة بسرعة.

## نصائح عملية
- اختر الأسئلة الأكثر تكرارًا من عملائك وضعها في هذا القسم.
- استخدم وصفًا مختصرًا وجذابًا لتشجيع المستخدمين على قراءة الأسئلة.
- أضف فيديو توضيحي إذا كان لديك شرح مرئي مفيد.
- اختر لون خلفية متناسق مع ألوان التطبيق.
