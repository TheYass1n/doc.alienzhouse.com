# Jameel Theme FAQ

**1. How do I change the main color scheme?**
Edit the `primary` color in your `tailwind.config.js` file.

**2. Can I add a custom font?**
Yes, update the `fontFamily` property in `tailwind.config.js` and include your font in the project.

**3. How do I enable product quick view?**
Ensure the quick view component is enabled in your theme settings or layout file.

**4. Is Jameel mobile-friendly?**
Absolutely. Jameel is designed with a mobile-first approach and uses responsive Tailwind classes.

**5. How do I update the Instagram feed?**
Replace the Instagram API key or widget code in the designated section of your homepage.
